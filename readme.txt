This plugin embeds the Trusted Shops Consent Manager (3rd Party), which you get as a customer of the Trusted Shops Legal Services.

== Description ==
**GERMAN CONCLUSION**
Dieses Plugin bettet den Consent-Manager von Trusted Shops ein, den man als Kunde der Trusted Shops Legal Services erhält. Das Plugin bindet dann das benutzerspezifische JavaScript ein, welches nur über die ID des Kunden verfügbar ist.
Aufgrund der Richtlinien für WordPress müssen Sie ein aktuelles WordPress System besitzen um dieses Plugin aus dem Store laden zu können.

**Hinweis zur Nutzungsbedingung:**
Sie müssen Kunde der Trusted Shops Legal Services sein um das Plugin nutzen zu können, da Sie eine Script-ID erhalten, welche nur Kunden vorbehalten ist.
Mehr Hinweise dazu finden Sie unter https://legal.trustedshops.com/

**Datenschutzrechtliche Hinweise:**
Das Javascript wird von einem **Drittanbieter** geladen, dabei handelt es sich um Usercentrix. Es wird lediglich das JavaScript angefragt, es findet kein weiterer Austausch statt. Der Dienst wird dazu genutzt, die Nutzerzustimmung für die Verarbeitung von Cookies für Nutzer zu vereinfachen und setzt Do-Not-Track Befehle je nach Zustimmung des Nutzers.
Weitere Informationen zum Drittanbieter finden Sie hier: https://usercentrics.com/de/

-----------------------------------------

**ENGLISH CONCLUSION**
This plugin embeds the Trusted Shops Consent Manager, which you get as a customer of the Trusted Shops Legal Services. The plugin then embeds the user-specific JavaScript, which is only available via the customers ID.

**Note on the condition of use:**
You must be a customer of Trusted Shops Legal Services to use the plugin, as you will receive a script ID which is reserved for customers only.
You can receive more detailed information on https://legal.trustedshops.com/

**Privacy notice:**
The JavaScript is loaded by a **third-party provider**, which is usercentrics. Only the JavaScript is requested, no further exchange takes place. The service is used to facilitate user consent for processing cookies for users and sets do-not-track commands depending on the users consent.
More information about the third party provider can be found here: https://usercentrics.com/de/
Privacy: https://legal.trustedshops.com/privacy
